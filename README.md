# Portfolio

My portfolio website. Welcome! :)

## Live

* https://cryptoisreal.org

## Run Scripts
```json
"start": "node server.js",
"dev": "react-scripts start",
"build": "react-scripts build",
"test": "react-scripts test",
"eject": "react-scripts eject"
```

## Built With
* NodeJS
* Express
* React
* RxJS
* Identicon.js
* Web3
* Sass
* IPFS (fleek.co hosting)
* Cloudflare
* Mobile friendly
